# My Salla Theme

Custom Salla e-commerce theme built with Twilight framework.

## Getting Started

### Prerequisites
- Node.js and pnpm installed
- Salla Partner account
- Salla CLI installed

### Installation

```bash
pnpm install
```

### Development

```bash
# Watch mode for development
pnpm watch

# Build for production
pnpm production
```

### Preview Theme

```bash
salla theme preview
```
